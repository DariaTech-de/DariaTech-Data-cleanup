# Enterprise Plan

This branch prepares the application for larger customer workloads.

## Scope

- background scan jobs
- progress status
- SQLite metadata
- versioned REST API
- report exports
- test and CI foundation
- documented path toward desktop installer and mobile clients

## Design principles

- local-first processing
- no cloud upload requirement
- review before applying changes
- auditability for customer work
- predictable behaviour on large folder trees
